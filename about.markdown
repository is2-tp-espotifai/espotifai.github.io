**Espotifai** es una plataforma moderna de streaming de música diseñada para que los usuarios puedan descubrir, reproducir y gestionar canciones, álbumes y playlists desde sus dispositivos móviles.
El sistema está construido sobre una arquitectura escalable de microservicios, donde cada servicio se encarga de un dominio específico: gestión y autenticación de usuarios, catálogo musical y notificaciones.

Las principales funcionalidades incluyen autenticación segura de usuarios (con soporte para login con Google), gestión de artistas y playlists, historial de reproducción y notificaciones en tiempo real sobre nuevos lanzamientos e interacciones sociales.
El backend utiliza FastAPI y Express.js para el desarrollo rápido de APIs, PostgreSQL para almacenamiento robusto de datos y Cloudflare R2 para la gestión eficiente de archivos multimedia.
La aplicación móvil está desarrollada con Expo y React Native, ofreciendo una experiencia fluida tanto en iOS como en Android.

Espotifai busca brindar una experiencia musical personalizada, confiable y agradable, combinando tecnologías modernas y buenas prácticas de arquitectura de software.
